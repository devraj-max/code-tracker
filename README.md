# 🚀 CodeTracker

**CodeTracker** is a full-stack web application built to help competitive programmers organize and track their coding journey. It provides a centralized dashboard to manage solved problems, monitor coding consistency, save personal notes, and stay updated with upcoming programming contests through real-time countdown timers.

## 🌐 Live Demo

👉 **https://code-tracker-beta.vercel.app/**

---

## ✨ Features

- 🔐 Secure user authentication using Appwrite
- 📊 Interactive dashboard for coding progress
- ✅ Add, edit, and delete coding problems
- 📝 Save personal notes and solution links for each problem
- 📌 Track problem details such as platform, difficulty, and status
- 🔥 Automatic coding streak calculation
- 📅 View upcoming programming contests
- ⏳ Real-time countdown timer for contests
- 📈 Visualize daily coding activity and consistency
- 🔍 Filter problems by platform, difficulty, and status
- 📱 Fully responsive design for desktop and mobile devices

## 🚀 Getting Started

### Clone the Repository

```bash
git clone https://github.com/your-username/code-tracker.git
```

### Navigate to the Project Directory

```bash
cd code-tracker
```

### Install Dependencies

```bash
npm install
```

### Create Environment Variables

Create a `.env` file in the project root and add the following:

```env
VITE_APPWRITE_ENDPOINT=YOUR_APPWRITE_ENDPOINT
VITE_APPWRITE_PROJECT_ID=YOUR_PROJECT_ID
VITE_APPWRITE_DATABASE_ID=YOUR_DATABASE_ID
VITE_APPWRITE_COLLECTION_ID=YOUR_PROBLEMS_COLLECTION_ID

```

Replace the above values with your own Appwrite credentials.

### Start the Development Server

```bash
npm run dev
```

The application will be available at:

```
http://localhost:5173
```
